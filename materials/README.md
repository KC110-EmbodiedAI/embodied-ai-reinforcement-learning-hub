# Materials

This directory is reserved for public course materials for students.

Suggested future structure:

```text
materials/
├── 00-course-guide/
├── 01-game-simulation/
├── 02-quadruped-control/
├── 03-sim2real-deployment/
├── 04-reflection-and-rubric/
└── media-index/
```

Suggested file types:

- learning route guide;
- lab manuals;
- environment setup notes;
- code template links or release notes;
- experiment record templates;
- assessment rubrics;
- redacted teaching demo notes;
- redacted media index.

Do not add the original PPT file, private screenshots, student information, personal data, or unpublished internal links.
